# سلام — من امیرحسین گنجی 👋

> توسعه‌دهندهٔ Python و علاقه‌مند به هوش مصنوعی و یادگیری ماشین — ساختن چیزهایی که می‌توانند «فکر» کنند.

من دانشجوی رشتهٔ علوم کامپیوتر مقیم ایران هستم. در حال حاضر روی پایان‌نامه‌ام کار می‌کنم با موضوع تشخیص فازهای Pump در بازارهای رمزارز با استفاده از یادگیری ماشین. علاقهٔ من به Python، AI/ML و توسعهٔ وب است.

---

## 🔭 وضعیت فعلی
- در حال انجام پایان‌نامه: **Crypto Pump Detection** — سیستم مبتنی بر ML برای شناسایی الگوهای pump-and-dump در داده‌های زمان-سری رمزارزها.
- مطالعه و تجربهٔ عملی با **LangChain** و معماری‌های RAG برای پیاده‌سازی برنامه‌های مبتنی بر LLM.
- تقویت مهارت‌ها در **Linux**, **Docker** و آماده‌سازی برنامه‌های Python برای محیط production.

## 📌 پروژه‌های من (مخازن عمومی)
در ادامه فهرست مخازن عمومی من همراه با توضیح کوتاه، لینک و نکات اجرای سریع آمده است.

### 🔍 Crypto Pump Detection (پایان‌نامه)
- توضیح: پایپ‌لاین ML برای شناسایی فازهای pump در داده‌های قیمت/حجم رمزارزها با استفاده از تحلیل زمان-سری و مدل‌های طبقه‌بندی.
- تکنولوژی‌ها: Python, pandas, scikit-learn, ویژگی‌های تکنیکال
- وضعیت: مخزن خصوصی — پس از دفاع پایان‌نامه عمومی خواهد شد.

### 🗂️ library-manager
- لینک: https://github.com/amirh-ganji/library-manager
- توضیح: نرم‌افزار دسکتاپ مدیریت کتابخانه با رابط کاربری Tkinter و دیتابیس SQLite؛ پروژهٔ دانشگاهی با امکانات مدیریت کتاب و کاربران (نقش مدیر/کاربر).

- تکنولوژی‌ها: Python, Tkinter, SQLite

### ✅ Todo-List (Frontend)
- لینک: https://github.com/amirh-ganji/Todo-List
- توضیح: اپلیکیشن Todo با HTML/CSS/Vanilla JS که به یک REST API (MockAPI) متصل می‌شود تا CRUD کامل انجام دهد.
- اجرا: باز کردن `index.html` در مرورگر؛ برای استفاده از API اختصاصی، مقدار `api_url` را در `script.js` جایگزین کنید.
- تکنولوژی‌ها: HTML, CSS, JavaScript

### 📰 news-scraper
- لینک: https://github.com/amirh-ganji/news-scraper
- توضیح: اسکریپت پایتون برای استخراج تیترها و لینک‌های بخش اقتصادی وب‌سایت Tasnim با استفاده از `requests` و `BeautifulSoup` و خروجی CSV.

- تکنولوژی‌ها: Python, BeautifulSoup, requests

### 🔢 handwritten-digit-recognition
- لینک: https://github.com/amirh-ganji/handwritten-digit-recognition
- توضیح: پروژهٔ یادگیری ماشین (Jupyter Notebook) که چهار الگوریتم طبقه‌بندی را روی دیتاست ارقام دست‌نویس (sklearn) مقایسه می‌کند؛ شامل پیش‌پردازش و نمایش معیارهای عملکرد.

- تکنولوژی‌ها: Python, scikit-learn, Jupyter
